a) Przejdź do pliku rekolekcje.js
b) Zmodyfikuj nagłówek (jeżeli trzeba)
c) Dodaj / usuń oraz zmodyfikuj zawartość wpisu rekolekcji

a) Przejdź do pliku aktualnosci.js
b) Zmodyfikuj nagłówek (jeżeli trzeba)
c) Dodaj / usuń oraz zmodyfikuj zawartość wpisu aktualności

4. Jak dodać nowy element w menu / linkach?
a) Przejdź do pliku nawigacja.js
b) Dodaj / usuń / zmodyfikuj element na liście menu
c) Dodaj / usuń / zmodyfikuj element na liście linki
d) Aby ukryć, ale nie usuwać, linku wystarczy przed wybranym wpisem dodać //

5. Jak dodać / usunąć plakaty ze strony głównej?
a) Przejdź do pliku plakaty.js
b) Dodaj / usuń wpisy z listy plakaty
c) Dodając nowy plakat upewnij się, że nowy plakat znajduje się pod ścieżką obrazy/plakaty oraz że nazwa pliku jest taka sama
d) Aby ukryć, ale nie usuwać plakatu z listy, wystarczy przed wybranym wpisem dodać //
e) Jeżeli wstawione plakaty nie mieszczą się w głównej sekcji strony - należy zmniejszyć wartość zmiennej "szerokosc" w pliku plakaty.js
f) Jeżeli plakaty są za małe, wartość zmiennej "szerokosc" można zwiększyć
